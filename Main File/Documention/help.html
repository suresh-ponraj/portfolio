Changing Fonts

If you would like to use a different font, copy the font link in Google Fonts page and paste it before closing head tag in your HTML files.

<link href="https://fonts.googleapis.com/css?family=Hind:400,500" rel="stylesheet">

Then you need to add the font name to your CSS styles.

:root {
	--heading-font-bold: 'func-font-bold';
	--heading-font-regu: 'func-font-regu';
	--para-font: 'Hind', sans-serif;
}

Then you need to add the font name to your CSS styles.

body {
	font-family: var(--heading-font);
}

Swiper Slider

All slider except work-smaple-main slider are created using Swiper Slider. You can change the Slider settings in assets/js/main.js under "Swiper Slider". Detailed information and options can be found in http://idangero.us/swiper/api/
